<br/>

## Quick Start

1. run `yarn add @mmstudio/widgets` or `npm i @mmstudio/widgets` install it.

2. import into project:

```jsx
import { MMProvider, CssBaseline } from '@mmstudio/widgets';

const Application = () => (
	<MMProvider>
		<CssBaseline /> // ---> Normalize styles
		<AppComponent /> // ---> Your App Component
	</MMProvider>
);
```
